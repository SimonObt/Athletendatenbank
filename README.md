# Athletendatenbank

Dies ist das Hauptprojekt-Verzeichnis für die Athletendatenbank.

## Projektstruktur

```
athletendatenbank/
├── features/              # Feature-Spezifikationen (PROJ-1-*.md, PROJ-2-*.md, ...)
├── src/                   # Source Code
│   ├── components/        # UI-Komponenten
│   ├── app/              # App-Router (Next.js) oder Haupt-App
│   └── lib/              # Utilities, Hooks, Helpers
├── docs/                  # Dokumentation
└── PROJECT_CONTEXT.md     # Projekt-Übersicht & Status
```

## Workflow

1. **Feature starten:**
   ```
   Lies agents/requirements-engineer.md und erstelle ein Feature für [Idee]
   ```

2. **Feature-Spec wird erstellt in:** `features/PROJ-X-feature-name.md`

3. **Architektur planen:**
   ```
   Lies agents/solution-architect.md und plane die Architektur für features/PROJ-X.md
   ```

4. **Implementieren:**
   ```
   Lies agents/frontend-dev.md und implementiere features/PROJ-X.md
   ```

5. **Testen:**
   ```
   Lies agents/qa-engineer.md und teste features/PROJ-X.md
   ```

## Getting Started

Siehe `PROJECT_CONTEXT.md` für aktuellen Status und nächste Schritte.
